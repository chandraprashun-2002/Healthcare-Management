# Healthcare Management API

A robust Django REST Framework (DRF) API for managing patients and clinic appointments.

## Features
- **Patient Management:** CRUD operations for patients.
- **Appointment System:** Appointment scheduling with patient links.
- **Analytics:** Summary statistics for clinic operations.
- **Search & Filter:** Advanced ORM filtering and search support.
- **Security:** Token-based Authentication and Staff-only write permissions.
- **Consistent Responses:** Standardized JSON format for all success/error responses.
